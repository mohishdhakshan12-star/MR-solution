# MR Solution - Web Portal

High-fidelity single-page site design for **MR Solution** — an electrical and hardware systems engineering firm.

## 🛠️ Stack
- **Structure**: Semantic HTML5
- **Styling**: Vanilla CSS3 (Dark, cinematic grid layout system)
- **Animations**: GSAP (GreenSock Animation Platform) + ScrollTrigger
- **Deployment**: Vercel
- **Version Control**: Git & GitHub

## 📁 File Structure
- `index.html` - Core markup and inline SVG assets.
- `styles.css` - Custom styling, typography, variables, responsive breakpoints, and accessibility.
- `app.js` - GSAP timeline triggers, stats counter, scrolling line draws, and form submissions.
- `.gitignore` - Standard configuration to ignore local OS/editor files.

## 🚀 Running Locally
You can run this project locally by opening `index.html` directly in any web browser, or by serving it using a lightweight local web server:

```bash
# Example: Using Python's built-in HTTP server
python -m http.server 8000
```
Then navigate to `http://localhost:8000`.

## 📦 Deployment via Vercel & GitHub

This portal is optimized for immediate deployment on **Vercel** with continuous deployment (CD) through **GitHub**. See the deployment guide in your terminal or prompt for detailed step-by-step instructions.
